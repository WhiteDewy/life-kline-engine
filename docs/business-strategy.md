# 占星人生节律产品 PRD v1.0（增长优先版）

## 产品定位一句话
一个“像算命一样好点、但不会让人上瘾和失控”的人生节律解释工具。

## 一、市场与用户真实行为分析（非理想化）

### 1.1 已有产品的真实运行逻辑
**测测 / 爱星盘的共同点（你必须承认它们做对的地方）**
*   **入口极其情绪化**：
    *   最近运气好不好？
    *   他是不是我的正缘？
    *   我什么时候才能顺？
*   **强即时反馈**：3–10 秒内必出结果
*   **模糊但可自我投射**：用户会主动“对号入座”
*   **轻度恐惧 + 希望并存**（这是转化的关键）

**它们的致命问题（你的机会）**
*   解释不可复用（看完即弃）
*   强预测导向 → 依赖 / 审核风险
*   用户只能“信或不信”，没有理解感
*   长期留存弱，只能靠频繁推新玩法

### 1.2 你的目标用户真实画像（去滤镜）
**核心用户（80% 收入来源）**
*   20–35 岁
*   情绪焦虑但理性尚存
*   对星座 / 玄学 半信半疑，但愿意花小钱买安慰

**真正想问的是：**
“是不是我有问题？还是我现在阶段不对？”

**⚠️ 注意：**
他们 **不想被说教**，也 **不想被吓**，
只想 **被解释清楚**。

## 二、产品核心策略（增长优先，不自嗨）

### 2.1 总体策略
用算命式问题吸引用户，用人生节律给答案，用画像体系承接复购。

**产品必须做到：**
1.  点进来像测测
2.  用起来不像测测
3.  用久了，比测测高级

## 三、核心产品结构（用户视角）
1.  入口问题（算命感）
    ↓
2.  基础盘生成（免费）
    ↓
3.  情绪共鸣式结论（免费）
    ↓
4.  节律解释 + 画像（付费）
    ↓
5.  持续订阅（人生阶段跟踪）

## 四、核心功能模块设计

### 4.1 首页 / 增长入口页
**用户心理状态**
我只是随便看看，但希望有点准。

**页面结构**
*   **大标题（强情绪）**：
    你最近是不是做什么都不太顺？
*   **副标题（去迷信）**：
    这不一定是运气问题，很多人只是走到了同一个人生阶段。
*   **CTA**：
    生成我的人生星盘

**⚠️ 禁忌：**
*   不出现“预测命运”
*   不出现“必然发生”

### 4.2 基础星盘生成（免费）
**功能**
*   日 / 月 / 上升
*   七大行星落座
*   简化解释（性格 + 行为倾向）

**文案原则**
*   只描述“你是怎样的”
*   不描述“你会发生什么”

**目的：**
建立信任：它好像真的懂我。

### 4.3 当前人生阶段（核心付费点）
**用户真实问题**
我现在卡住，是不是我不行？

**交付内容**
*   当前阶段名称（非吉凶）
*   阶段关键词（3–5 个）
*   常见感受（高度共情）
*   为什么很多人在这个阶段都会难受

**示例文案：**
> 你现在经历的，并不是低谷，
> 而是一个“旧模式失效、新模式尚未建立”的阶段。

### 4.4 人生节律曲线（原“人生K线”）
**命名策略（过审 & 心理安全）**
*   **官方名**：人生节律曲线
*   **对用户展示可用**：
    *   人生节奏图
    *   成长周期图

**展示原则**
*   只展示 **强弱变化**
*   **不标注具体事件**
*   **不出现“吉 / 凶”**

### 4.5 人生画像系统（复购核心）
**画像类型**
*   桃花画像（关系模式）
*   正缘画像（吸引类型，不是具体人）
*   事业画像（工作方式 +优势区）
*   学业画像
*   人际关系画像

**用户为什么愿意买**
原来不是我不行，是我用错了方式。

## 五、商业化设计（符合真实付费心理）

| :--- | :--- | :--- |

## 六、与测测 / 爱星盘的本质差异

| 维度 | 它们 | 你 |
| :--- | :--- | :--- |
| 入口 | 算命 | 像算命 |
| 核心 | 预测 | 解释 |
| 情绪 | 刺激 | 安抚 |
| 留存 | 靠上瘾 | 靠理解 |
| 风险 | 高 | 可控 |

## 七、阶段性产品路线图

### Phase 1（活下来）
*   强入口
*   单一阶段付费
*   少功能，高转化

### Phase 2（站住）
*   画像系统
*   节律订阅

### Phase 3（做大）
*   合盘 / 关系节律
*   家庭 / 亲子模块

## 八、成功指标（不自嗨）
*   首页 → 生成盘 转化 ≥ 40%
*   免费 → 首单 ≥ 8%
*   首单 → 二次付费 ≥ 30%

**用户评价关键词：**
*   “被理解”
*   “说清楚了”

**产品底线原则：**
我们不贩卖恐惧，不制造依赖，
只做人生阶段的解释器。
